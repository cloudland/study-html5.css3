定位属性: position

            定位 = 定位模式 + 偏移
            定位模式用于指定一个元素在文档中的定位方式, 边偏移则决定了该元素的最终位置.
定位模式
值 语义 用处
static 静态定位 很少用, 可以理解为标准流
relative 相对定位 在移动位置后, 相对于原来的位置并占有原来位置
absolute 绝对定位 不占位。相对于父级元素来定位, 父元素没有定位或没有父级, 以浏览器为准定位。
fixed 固定定位 不占位。固定于浏览器可视区位置。
边偏移

定位元素移动到最终位置。有 top、bottom、left、right 4个属性。

            元素加了定位, 可以直接设置宽、高。如果不给定, 默认内容大小。
